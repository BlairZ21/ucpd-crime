We filed a California Public Records Act request with UCPD for incident-level crime data and reviewed daily incident logs from January 2010 to September 2015.

To analyze crime by incident type, we created a mapping of incident descriptions from the raw data (such as “VANDALISM”) to one of three categories: property crime, violent crime and quality-of-life crime. You can view that mapping here. Although we tried to use commonly accepted definitions of these crime categories, the classification necessarily involved some subjective judgement about to which category a particular crime type belongs.

Many of the logged events were security checks, alarms or escorts. We excluded those incidents from our analysis. In all, we analyzed 13,046 reports.

The data contains a location field — sometimes an address, but often an abbreviation, such as “BAN/TELE” for the intersection of Bancroft Way and Telegraph Avenue, or a commonly-used name, such as “GRINNELL PATH.” To put these incidents on a map, we had to assign each point a latitude and longitude.

Because geocoding services don't recognize many of these canonical names, we had to manually construct a mapping of incident locations to latitude and longitude, which you can view here. After manually checking the list, we were able to geocode 11,198 incidents.

All of the data and code used for this project is available to the public.
